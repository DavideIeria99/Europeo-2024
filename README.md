# simulazione Europeo 2024

## Descrizione del Progetto

Questo progetto è una web application sviluppata con Next.js che simula il torneo degli Europei di calcio 2024. L'applicazione consente agli utenti di simulare le partite, visualizzare i risultati generati, consultare le classifiche aggiornate e seguire l'andamento del torneo come se fosse reale.

## Pacchetti utilizzati

- **Next.js** – Framework React per lo sviluppo web
- **React** – Libreria per la creazione di interfacce utente
- **Tailwind CSS** – Framework utility-first per la stilizzazione
- **Prisma** – Orm database
- **ContextApi** – Gestione dello stato globale
- **react-icons** – Icone vettoriali per React
- **sacdn**- per la creazione delle tabelle.
